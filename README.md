# Wordly Dictionary

## Description

A Single Page Application (SPA) dictionary built using HTML, CSS, and JavaScript.  Allows users to search for words and view definitions, pronunciations, and synonyms (where available) fetched from a dictionary API.

## Features Implemented

*   Word search functionality.
*   Display of word definitions.
*   Display of pronunciations (if available).
*   Display of synonyms (if available).
*   Error handling for:
    *   Empty input.
    *   Word not found.
    *   API errors.
*   Dynamic updating of the page (no page refresh).

## Instructions on How to Run the Project

1.  Clone the repository to your local machine: `git clone <your-repository-url>`
2.  Open `index.html` in your web browser.
